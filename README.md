Wistia Upload Component
=====

This is an Angular component for uploading videos to [Wistia](https://wistia.com/)

### Installation

Clone or download the repository and then:

```sh
$ npm install && npm start && nodemon
```

### Running Tests

```sh
$ npm test
```
